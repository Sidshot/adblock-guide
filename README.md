# Adblock Guide

A comprehensive, beginner-friendly guide to blocking ads on any device.

**Live Site**: https://sidshot.github.io/adblock-guide/

## Features

- Step-by-step guides for Windows, macOS, iOS, and Android
- Modern, responsive design with dark mode
- Troubleshooting section for common issues
- FAQ answering common questions
- Zero dependencies - Pure HTML, CSS, and JavaScript

## Platforms Covered

| Platform | Recommended Tool | Browser Support |
|----------|-----------------|-----------------|
| Windows | uBlock Origin | Firefox, Chrome, Edge |
| macOS | uBlock Origin / Lite | Safari, Firefox, Chrome |
| iOS | AdGuard | Safari |
| Android | uBlock Origin / AdGuard | Firefox / System-wide |

## Local Development

Open `index.html` in your browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
